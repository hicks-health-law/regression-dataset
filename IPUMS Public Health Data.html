The rule of thumb for VIF determination of multicollinearity is as follows:

Table from: https://www.statology.org/multicollinearity-in-r/
VIF Result
1 There is no correlation between a given predictor variable and any other predictor variables in the model.
Between 1 & 5 There is moderate correlation between a given predictor variable and other predictor variables in the model.
Greater than 5 There is severe correlation between a given predictor variable and other predictor variables in the model.
